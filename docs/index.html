FreeAgent Design Language.

Get from prototype to product super fast.

Principles

We should reference and respect these principles when creating new design patterns at FreeAgent.

Basics

The smallest core building blocks of any design. They can not be broken down any further.

Components

Interface patterns that fall on the moderately complex side of the spectrum.

Templates

Templates document the layout and structure of a section or the entirety of an interface.

Features

The interfaces that, when working together, allow users to accomplish a task or solve a problem.

Applications

The entry point to the design system where we document the various products built using the design system.
