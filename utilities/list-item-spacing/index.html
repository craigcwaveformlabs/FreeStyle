list-item-spacing

View SCSS source

The list-item-spacing utility lets us control the space between list items.

Values

  • tight
  • default
  • loose
  • x-loose
  • xx-loose

Examples

  • Default list item spacing
  • Default list item spacing
  • Default list item spacing

  <li>Default list item spacing</li>
  <li>Default list item spacing</li>
  <li>Default list item spacing</li>
</ul>
.MyList li { @include list-item-spacing(default); }
.MyList li {
